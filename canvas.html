    Ajuda

    Desenhos
    1. Linha: Para desenhar uma linha, clique no botão com o nome "LINHA", em seguida clique em um ponto no canvas, e sem soltar o clique arraste para o segundo ponto.
    2. Círculo: Para desenhar um círculo, clique no botão com o nome "CÍRCULO", em seguida clique em um ponto no canvas, e sem soltar o clique arraste até um segundo ponto, e a partir do segundo ponto será calculado o valor do raio do círculo.
    3. Retângulo:Para desenhar um retângulo, clique no botão com o nome "RETÂNGULO", em seguida clique em um ponto no canvas, e sem soltar arraste até outro ponto montando a diagonal principal do retângulo.
    4. Triângulo: Para desenhar um triângulo, clique no botão com o nome "TRIÂNGULO", em seguida clique em três pontos no canvas, onde cada um corresponde a uma ponta do objeto.
    5. Limpar Tela: Para limpar a tela, clique no botão com o nome "LIMPAR" e confirme.
    Transformações
    1. Translação: Para realizar a translação, clique no botão com o nome "TRANS.", em seguida informe o índice de qual objeto da lista será transladado, e por fim informe os valores de TX e TY que serão utilizados.
    2. Mudança de escala: Para realizar a mudança de escala, clique no botão com o nome "ESCALA", em seguida informe o índice de qual objeto da lista terá sua escala modificada, e por fim informe os valores de SX e SY.
    3. Rotação: Para realizar a rotação, clique no botão com o nome "ROTAÇÃO", em seguida informe o índice de qual objeto da lista será rotacionado, e por fim informe o ângulo de rotação (em graus, por exemplo 90).
    Comandos
    1. Linha: O comando para desenhar uma linha é line(x0, y0, x1, y1), onde os dois primeiros valores correspondem ao primeiro ponto da reta e o os dois últimos correspondem ao segundo ponto da reta.
    2. Círculo: O comando para desenhar um círculo é circle(x0, y0, radius), onde os dois primeiros valores correspondem ao centro do círculo e o parâmetro radius corresponde ao raio do mesmo.
    3. Retângulo: O comando para desenhar um retângulo é rect(x0, y0, x1, y1), onde os dois primeiros valores corres a um primeiro ponto e os últimos dois valores correspodem ao ponto que completa uma diagonal com o primeiro ponto.
    4. Triângulo: O comando para desenhar um triângulo é triangle(x0, y0, x1, y1, x2, y2), onde a cada xi e yi corresponde a um canto do triângulo.
    5. Limpar Tela: O comando para limpar a tela é clear(), apos digitá-lo é necessário confirmar a ação.
    6. Aumentar zoom: O comando para aumentar o zoom é zoomIn(), ele irá duplicar a escala dos objetos.
    7. Reduzir zoom:O comando para diminuir o zoom é zoomOut(), ele irá reduzir pela metade a escala dos objetos.
    8. Redefinir: O comando para redefinir o zoom é zoomReset(), ele irá deixar os objetos como eram antes da aplicação do aumento ou redução de zoom.
    9. Seleção de objetos: Para selecionar objetos use o comando select(indice), onde o parâmetro é o índice do objeto em relação à lista de objetos.
    10. Deselecionar objetos: Para deselecionar objetos basta usar o comando deselect(indece), onde o parâmetro é o índice do objeto em relação à lista de objetos.
    Zoom
    1. Aumentar zoom: Para aumentar o zoom clique no botão que possui a lupa com o sinal "+".
    2. Reduzir zoom: Para diminuir o zoom clique no botão que possui a lupa com o sinal "-".
    3. Redefinir: Para redefinir o canvas clique no ícone com os dois quadrados.
    Observações
    1. Os pontos mostrados na lista de objetos de um retângulo são os pontos da diagonal principal do mesmo.
    Sair: Para sair desta tela aperte ESC.
